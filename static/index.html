API Documentation

Simple REST API endpoints for address processing

POST /api/v1/parse

Parse an address into structured components

Request Body:

{
  "address": "123 Main St, New York, NY 10001",
  "language": "en",
  "country": "US"
}

cURL Example:

curl -X POST https://libpostal.pendejo.dev/api/v1/parse \
  -H "Content-Type: application/json" \
  -d '{
    "address": "123 Main St, New York, NY 10001",
    "language": "en",
    "country": "US"
  }'

Response:

{
  "success": true,
  "data": {
    "original": "123 Main St, New York, NY 10001",
    "components": {
      "house_number": "123",
      "road": "Main St",
      "city": "New York",
      "state": "NY",
      "postcode": "10001"
    }
  },
  "meta": {
    "processing_time_ms": 15,
    "request_id": "req-123",
    "api_version": "1.0",
    "timestamp": "2025-01-01T00:00:00Z"
  }
}

POST /api/v1/normalize

Normalize and expand address abbreviations

Request Body:

{
  "address": "123 Main St",
  "level": "medium"
}

cURL Example:

curl -X POST https://libpostal.pendejo.dev/api/v1/normalize \
  -H "Content-Type: application/json" \
  -d '{
    "address": "123 Main St",
    "level": "medium"
  }'

Response:

{
  "success": true,
  "data": {
    "original": "123 Main St",
    "expansions": [
      "123 main street",
      "123 main st"
    ],
    "expansion_count": 2
  },
  "meta": {
    "processing_time_ms": 8,
    "request_id": "req-124",
    "api_version": "1.0",
    "timestamp": "2025-01-01T00:00:00Z"
  }
}
